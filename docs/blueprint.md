# **App Name**: TremplinVideoLab

## Core Features:

- Homepage Carousel: Display the three latest videos in a carousel format on the homepage.
- Video Listing Page: Display all videos on a dedicated page.
- Admin Dashboard: Enable administrators to modify the homepage text and upload new videos via a protected admin interface.
- Video Upload: Allow admins to upload videos, which are then stored.
- Homepage Text Editor: Allow admins to edit the homepage content/text.
- Database Storage: All uploaded videos are saved to a Postgres database.
- Admin Login: Secure login system for accessing the admin panel.
- User Management: Admin page for managing user accounts and permissions.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of trust and sophistication.
- Background color: Light Gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Vivid Orange (#FF9800) for highlights and interactive elements to draw user attention.
- Body font: 'PT Sans', a humanist sans-serif for readability and a modern touch.
- Headline font: 'Space Grotesk', a sans-serif with a contemporary, techy look.
- Use modern, minimalist icons for navigation and actions.
- Employ a clean, grid-based layout with ample whitespace to improve content scannability.
- Incorporate subtle transitions and animations for a smooth and engaging user experience.
